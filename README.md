# Dashboard Video Tool

This project is a tech test based on a design provided by Be Master.

Used tools:

- React 18 + Vite + TS
- React Router
- Tailwind CSS
- Font Awsome Icons

## Installation

If you want to explore the project, just clone this repo, navigate to the route and run `npm i`, then, run `npm run dev`.
